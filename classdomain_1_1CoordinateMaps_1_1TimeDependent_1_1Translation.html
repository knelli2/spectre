Detailed Description

template<size_t Dim>
class domain::CoordinateMaps::TimeDependent::Translation< Dim >

Translation map defined by \(\vec{x} = \vec{\xi}+\vec{T}(t)\).

The map adds a translation, \(\vec{T}(t)\), to the coordinates \(\vec{\xi}\), where \(\vec{T}(t)\) is a FunctionOfTime.
