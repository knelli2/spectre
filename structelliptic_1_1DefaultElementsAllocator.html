Detailed Description

template<size_t Dim>
struct elliptic::DefaultElementsAllocator< Dim >

A Parallel::protocols::ArrayElementsAllocator that creates array elements to cover the initial computational domain.

An element is created for every element ID in every block, determined by the initial_element_ids function and the option-created domain::Tags::Domain and domain::Tags::InitialRefinementLevels. The elements are distributed on processors using the domain::BlockZCurveProcDistribution. In both cases, an unordered set of size_ts can be passed to the allocate_array function which represents physical processors to avoid placing elements on.
